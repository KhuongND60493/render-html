Icons

A font icons and svg icons from different open source collections for you to use anywhere in the design of your project.

Feather Icons

Feather is a collection of simply beautiful open source icons. Each icon is designed on a 24x24 grid with an emphasis on simplicity, consistency and usability. https://feathericons.com/

For a complete list of feather icons, please check Feather Icons Official Website


        


        
Ionicons

        
Premium designed icons for use in web, iOS, Android, and desktop apps. Support for SVG and web font. Completely open source, MIT licensed and built by the Ionic Framework team.


        
For a complete list of ionicons icons, please check Ionicons Official Website

Typicons

        
Typicons are free-to-use vector icons embedded in a webfont for easy use in any UI, whether it be on the web or in a native app.


        
For a complete list of Typicon icons, please check Typicon Official Website
